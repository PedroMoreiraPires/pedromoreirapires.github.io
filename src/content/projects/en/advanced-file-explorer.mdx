# Advanced File Explorer

This project is a complete solution for file management in corporate or personal environments. Developed with a robust architecture using **Java** and **Spring Boot** in the backend, and **Angular** in the frontend, it offers a smooth and secure user experience.

## Key Features

- **Advanced Search**: Allows filtering files by name, extension, modification date, and size.
- **Preview**: Support for viewing PDF documents, images, and text files directly in the browser.
- **Permission Management**: Granular role and permission system to control who can view, edit, or delete files.
- **Intuitive Interface**: Modern and responsive design adapted to different devices.

## Technologies Used

- **Backend**: Java 17, Spring Boot 3.
- **Frontend**: Angular 16, TypeScript, RxJS.
- **Database**: MySQL for file metadata.
- **Storage**: Integration with local file system and S3 support.

This explorer not only facilitates directory navigation but also optimizes workflow by integrating quick visualization and editing tools.
