# Red Social Sostenible

Una plataforma innovadora que conecta a personas comprometidas con el medio ambiente. El objetivo principal es fomentar la recolección de residuos y la organización de eventos comunitarios de limpieza.

## ¿Qué ofrece la plataforma?

- **Organización de Eventos**: Herramientas para crear, difundir y gestionar eventos de limpieza y concienciación ambiental.
- **Sistema de Recompensas**: Gamificación que premia a los usuarios activos con puntos canjeables por productos sostenibles o descuentos.
- **Comunidad**: Foros y grupos para compartir consejos, logros y coordinar acciones locales.
- **Mapa Interactivo**: Visualización de puntos de recolección, eventos cercanos y zonas que necesitan atención.

## Arquitectura

- **Frontend**: Angular para una interfaz dinámica y reactiva.
- **Backend**: Node.js con Express para una API rápida y escalable.
- **Base de Datos**: PostgreSQL con extensiones espaciales (PostGIS) para las funcionalidades de mapa.

Este proyecto busca crear un impacto positivo real, utilizando la tecnología para movilizar a la comunidad en pro de un planeta más limpio.
