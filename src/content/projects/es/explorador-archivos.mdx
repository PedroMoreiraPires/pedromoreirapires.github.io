# Explorador de Archivos Avanzado

Este proyecto es una solución completa para la gestión de archivos en entornos corporativos o personales. Desarrollado con una arquitectura robusta utilizando **Java** y **Spring Boot** en el backend, y **Angular** en el frontend, ofrece una experiencia de usuario fluida y segura.

## Características Principales

- **Búsqueda Avanzada**: Permite filtrar archivos por nombre, extensión, fecha de modificación y tamaño.
- **Previsualización**: Soporte para visualizar documentos PDF, imágenes y archivos de texto directamente en el navegador.
- **Gestión de Permisos**: Sistema de roles y permisos granular para controlar quién puede ver, editar o eliminar archivos.
- **Interfaz Intuitiva**: Diseño moderno y responsivo adaptado a diferentes dispositivos.

## Tecnologías Utilizadas

- **Backend**: Java 17, Spring Boot 3.
- **Frontend**: Angular 16, TypeScript, RxJS.
- **Base de Datos**: MySQL para metadatos de archivos.
- **Almacenamiento**: Integración con sistema de archivos local y soporte para S3.

Este explorador no solo facilita la navegación por directorios, sino que también optimiza el flujo de trabajo al integrar herramientas de visualización y edición rápida.
