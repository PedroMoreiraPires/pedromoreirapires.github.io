# Gestión Interna para Bares y Restaurantes

Solución todo en uno para la administración eficiente de establecimientos gastronómicos. Desde el control de stock hasta la gestión de turnos de empleados, esta aplicación centraliza todas las operaciones críticas.

## Módulos del Sistema

- **Inventario en Tiempo Real**: Control detallado de ingredientes y bebidas, con alertas de stock bajo.
- **Gestión de Pedidos**: Sistema para tomar comandas desde mesas o barra, enviándolas directamente a cocina.
- **Administración de Personal**: Gestión de horarios, turnos, nóminas y rendimiento de los empleados.
- **Panel Administrativo**: Estadísticas de ventas, platos más populares y rendimiento financiero.

## Tecnologías

- **Backend**: Java con Spring Boot para una lógica de negocio segura y escalable.
- **Frontend**: React para una experiencia de usuario ágil e interactiva.
- **Base de Datos**: MongoDB para manejar datos flexibles como menús y configuraciones de pedidos.

Diseñada para optimizar los tiempos y reducir las pérdidas, esta herramienta es esencial para cualquier gerente de bar o restaurante que busque modernizar su gestión.
