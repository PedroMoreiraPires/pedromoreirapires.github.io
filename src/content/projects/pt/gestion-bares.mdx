# Gestão Interna para Bares e Restaurantes

Solução completa para a administração eficiente de estabelecimentos gastronômicos. Desde o controle de estoque até a gestão de turnos de funcionários, esta aplicação centraliza todas as operações críticas.

## Módulos do Sistema

- **Inventário em Tempo Real**: Controle detalhado de ingredientes e bebidas, com alertas de baixo estoque.
- **Gestão de Pedidos**: Sistema para anotar pedidos de mesas ou do balcão, enviando-os diretamente para a cozinha.
- **Administração de Pessoal**: Gestão de horários, turnos, folhas de pagamento e desempenho dos funcionários.
- **Painel Administrativo**: Estatísticas de vendas, pratos mais populares e desempenho financeiro.

## Tecnologias

- **Backend**: Java com Spring Boot para uma lógica de negócios segura e escalável.
- **Frontend**: React para uma experiência de usuário ágil e interativa.
- **Banco de Dados**: MongoDB para lidar com dados flexíveis como cardápios e configurações de pedidos.

Projetada para otimizar o tempo e reduzir perdas, esta ferramenta é essencial para qualquer gerente de bar ou restaurante que busca modernizar sua gestão.
