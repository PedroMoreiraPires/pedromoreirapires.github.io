# Explorador de Arquivos Avançado

Este projeto é uma solução completa para a gestão de arquivos em ambientes corporativos ou pessoais. Desenvolvido com uma arquitetura robusta usando **Java** e **Spring Boot** no *backend*, e **Angular** no *frontend*, oferece uma experiência de usuário fluida e segura.

## Funcionalidades Chave

- **Busca Avançada**: Permite filtrar arquivos por nome, extensão, data de modificação e tamanho.
- **Pré-visualização (Preview)**: Suporte para visualização de documentos PDF, imagens e arquivos de texto diretamente no navegador.
- **Gestão de Permissões**: Sistema granular de funções e permissões para controlar quem pode visualizar, editar ou excluir arquivos.
- **Interface Intuitiva**: Design moderno e responsivo, adaptado a diferentes dispositivos.

## Tecnologias Utilizadas

- **Backend**: Java 17, Spring Boot 3.
- **Frontend**: Angular 16, TypeScript, RxJS.
- **Banco de Dados**: MySQL para metadados de arquivos.
- **Armazenamento**: Integração com sistema de arquivos local e suporte S3.

Este explorador não apenas facilita a navegação de diretórios, mas também otimiza o fluxo de trabalho ao integrar ferramentas de visualização e edição rápidas.
