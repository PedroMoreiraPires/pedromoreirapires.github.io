# Rede Social Sustentável

Uma plataforma inovadora que conecta pessoas comprometidas com o meio ambiente. O objetivo principal é incentivar a coleta de resíduos e a organização de eventos comunitários de limpeza.

## O que a Plataforma Oferece?

- **Organização de Eventos**: Ferramentas para criar, divulgar e gerenciar eventos de limpeza e conscientização ambiental.
- **Sistema de Recompensas**: Gamificação que premia usuários ativos com pontos resgatáveis por produtos sustentáveis ou descontos.
- **Comunidade**: Fóruns e grupos para compartilhar dicas, conquistas e coordenar ações locais.
- **Mapa Interativo**: Visualização de pontos de coleta, eventos próximos e áreas que precisam de atenção.

## Arquitetura

- **Frontend**: Angular para uma interface dinâmica e reativa.
- **Backend**: Node.js com Express para uma API rápida e escalável.
- **Banco de Dados**: PostgreSQL com extensões espaciais (PostGIS) para as funcionalidades de mapa.

Este projeto busca criar um impacto positivo real, utilizando a tecnologia para mobilizar a comunidade em prol de um planeta mais limpo.
